## 🎥 视频功能详解

### 1. 视频播放器
**功能**：
- 嵌入式 YouTube 播放器
- 全屏播放支持
- 自动播放控制
- 一键在 YouTube 打开
- 学习提示显示

**使用方法**：
- 在内容目录点击任意视频
- 播放器自动弹出
- 支持所有 YouTube 功能

### 2. 视频收藏系统
**功能**：
- 一键收藏/取消收藏
- 收藏列表管理
- 本地存储（localStorage）
- 显示收藏日期
- 快速播放收藏的视频

**使用方法**：
- 播放视频时点击❤️图标
- 在应用页面的 "Favorite videos" 标签查看
- 点击视频卡片即可播放

### 3. 视频搜索和管理（需要 API Key）
**功能**：
- 关键词搜索 YouTube 视频
- URL 直接添加视频
- 自动获取视频信息
- 缩略图预览
- 一键复制配置代码

**使用方法**：
1. 访问 `/admin` 管理页面
2. 搜索或粘贴 YouTube URL
3. 点击"复制代码"
4. 粘贴到 `lib/content-data.ts`

## 🎯 使用流程

### 完整学习流程
1. **浏览视频** → 访问 `/catalog`
2. **播放视频** → 点击视频卡片
3. **收藏视频** → 点击❤️图标
4. **导入文本** → 访问 `/app`，粘贴文本
5. **学习单词** → 点击单词查看翻译
6. **保存单词** → 点击书签图标
7. **听力练习** → 访问 `/phrasepump`
8. **查看统计** → 访问 `/stats`
9. **导出 Anki** → 在词汇列表点击导出

### 视频管理流程（可选）
1. **获取 API Key** → Google Cloud Console
2. **配置环境变量** → `.env.local`
3. **访问管理页面** → `/admin`
4. **搜索视频** → 输入关键词
5. **复制代码** → 点击按钮
6. **添加到项目** → 编辑 `content-data.ts`
