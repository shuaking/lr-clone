# PWA Icons

PWA 需要以下图标文件：

## 必需图标

1. **icon-192x192.png** (192x192 像素)
   - 用于 Android 主屏幕图标
   - 用于应用启动画面

2. **icon-512x512.png** (512x512 像素)
   - 用于高分辨率设备
   - 用于应用商店展示

3. **favicon.ico** (可选)
   - 浏览器标签页图标

## 设计建议

- 使用品牌色 #7c9cff (蓝紫色)
- 背景色 #0a0e1a (深色)
- 简洁的 "LR" 字母标识
- 确保在深色和浅色背景下都清晰可见

## 生成工具

可以使用以下工具生成 PWA 图标：
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## 临时方案

在正式图标准备好之前，可以使用纯色占位符：
- 192x192: 蓝紫色背景 + 白色 "LR" 文字
- 512x512: 同样设计，更高分辨率
