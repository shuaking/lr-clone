# 高级词典和机器翻译功能使用指南

## 📚 功能概述

本项目现已集成高级词典和机器翻译功能，提供更丰富的语言学习体验。

### 新增功能

#### 1. 机器翻译服务
- **多翻译源支持**：DeepL、Google Translate、LibreTranslate、MyMemory
- **自动降级**：当首选服务失败时，自动尝试其他服务
- **混合方案**：免费服务 + 可选付费服务
- **智能缓存**：减少 API 调用，提升响应速度

#### 2. 高级词典服务
- **多词典源**：Free Dictionary API、Tatoeba 例句库、词形变化识别
- **详细定义**：词性、释义、同义词、反义词
- **真人发音**：多种口音的发音音频（US、UK、AU）
- **丰富例句**：来自 Tatoeba 的真实例句，带翻译
- **词形变化**：自动识别单词的原形和各种变形

## 🚀 快速开始

### 1. 访问设置页面

1. 打开应用：访问 `/app`
2. 点击左侧导航栏的"设置"按钮
3. 进入 API 配置界面

### 2. 配置翻译 API（可选）

#### 免费方案（无需配置）
默认使用 LibreTranslate 和 MyMemory，无需任何配置即可使用。

#### 付费方案（高质量翻译）

**DeepL API**
1. 访问 [DeepL Pro API](https://www.deepl.com/pro-api) 注册账户
2. 获取 API Key
3. 在设置页面输入 API Key
4. 点击"测试"按钮验证连接
5. 免费版使用 `https://api-free.deepl.com/v2`
6. 付费版使用 `https://api.deepl.com/v2`

**Google Translate API**
1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 创建项目并启用 Cloud Translation API
3. 创建 API Key
4. 在设置页面输入 API Key
5. 点击"测试"按钮验证连接

**LibreTranslate（自托管）**
1. 部署自己的 LibreTranslate 实例
2. 在设置页面输入服务器 URL
3. 如果需要，输入 API Key

### 3. 配置词典设置

在设置页面的"词典配置"部分：

- **启用的词典源**：选择要使用的词典提供商
  - Free Dictionary API：英语定义和发音
  - Tatoeba：多语言例句
  - 词形变化识别：英语词形分析

- **最大例句数量**：设置显示的例句数量（1-20）

## 💡 使用方法

### 查看单词详细信息

1. 在视频学习页面点击任意单词
2. 弹出的词典面板会显示：
   - **翻译**：使用配置的翻译服务
   - **发音**：多种口音的音频（如果可用）
   - **释义**：详细的词性和定义
   - **例句**：真实语境中的使用示例
   - **词形变化**：原形和各种变形

### 切换显示内容

词典面板有三个标签页：
- **词典**：翻译、定义、例句、词形变化
- **语法**：语法分析和解释
- **文化**：文化背景和注释

### 播放发音

- 点击发音按钮播放不同口音的音频
- 支持 US（美式）、UK（英式）、AU（澳式）等

### 展开/折叠内容

- 点击"释义"、"例句"、"词形变化"标题可展开或折叠内容
- 默认展开释义，折叠例句和词形变化

## 🔧 高级配置

### 翻译服务优先级

在设置页面可以选择首选翻译服务：
- **自动选择**：按优先级尝试（DeepL > Google > LibreTranslate > MyMemory）
- **DeepL**：优先使用 DeepL（需要 API Key）
- **Google Translate**：优先使用 Google（需要 API Key）
- **LibreTranslate**：优先使用 LibreTranslate
- **MyMemory**：优先使用 MyMemory

### 自动降级

启用"自动降级"后：
- 当首选服务失败时，自动尝试下一个可用服务
- 确保翻译功能始终可用
- 推荐保持启用状态

### 缓存机制

- 翻译结果缓存 1 小时
- 词典结果缓存 24 小时
- 减少 API 调用次数
- 提升响应速度

## 📊 API 使用限制

### 免费服务

**LibreTranslate（公共实例）**
- 限制：可能有速率限制
- 质量：中等
- 语言：12+ 种语言

**MyMemory**
- 限制：每天 1000 次请求（匿名）
- 质量：中等
- 语言：13+ 种语言

### 付费服务

**DeepL API Free**
- 限制：每月 500,000 字符
- 质量：最高
- 语言：28+ 种语言
- 价格：免费（有限额）

**DeepL API Pro**
- 限制：按使用量计费
- 质量：最高
- 语言：28+ 种语言
- 价格：$5.49/月起

**Google Translate API**
- 限制：按使用量计费
- 质量：高
- 语言：100+ 种语言
- 价格：$20/百万字符

## 🛠️ 故障排除

### 翻译失败

**问题**：显示"翻译失败"或"翻译服务暂时不可用"

**解决方案**：
1. 检查网络连接
2. 验证 API Key 是否正确
3. 检查 API 配额是否用完
4. 尝试切换到其他翻译服务
5. 启用"自动降级"功能

### 词典数据不完整

**问题**：某些单词缺少定义或例句

**解决方案**：
1. 确保启用了所有词典源
2. Free Dictionary API 仅支持英语
3. Tatoeba 例句可能不是所有单词都有
4. 某些专业术语可能缺少数据

### 发音音频无法播放

**问题**：点击发音按钮没有声音

**解决方案**：
1. 检查浏览器音量设置
2. 确保浏览器允许音频播放
3. 某些单词可能没有音频文件
4. 尝试使用 Web Speech Synthesis 作为后备

### API 连接测试失败

**问题**：点击"测试"按钮显示连接失败

**解决方案**：
1. 检查 API Key 是否正确
2. 检查 API URL 是否正确
3. 确认 API 服务正常运行
4. 检查防火墙或代理设置
5. 查看浏览器控制台的错误信息

## 🔐 隐私和安全

### 数据存储

- API Key 存储在浏览器 localStorage
- 不会发送到任何第三方服务器
- 仅在调用对应 API 时使用

### API 调用

- 翻译和词典查询直接调用对应的 API
- 不经过中间服务器
- 遵循各 API 提供商的隐私政策

### 建议

- 不要在公共设备上保存 API Key
- 定期更换 API Key
- 监控 API 使用量，避免超额费用

## 📝 开发者信息

### 架构

```
lib/
├── translation/              # 翻译服务
│   ├── types.ts             # 类型定义
│   ├── translation-service.ts  # 主服务
│   └── providers/           # 翻译提供商
│       ├── deepl.ts
│       ├── google.ts
│       ├── libretranslate.ts
│       ├── mymemory.ts
│       └── mock.ts
├── dictionary/              # 词典服务
│   ├── types.ts            # 类型定义
│   ├── dictionary-service.ts  # 主服务
│   └── providers/          # 词典提供商
│       ├── free-dictionary.ts
│       ├── tatoeba.ts
│       └── lemmatization.ts
└── stores/
    └── api-config-store.ts  # API 配置存储
```

### 扩展

要添加新的翻译或词典提供商：

1. 在 `providers/` 目录创建新文件
2. 实现 `ITranslationProvider` 或 `DictionaryProvider` 接口
3. 在主服务中注册新提供商
4. 在 API 设置界面添加配置选项
