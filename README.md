# Language Reactor Clone

一个语言学习 Web 应用，实现了 Language Reactor 的**大部分核心功能**。

> **重要说明**: 本项目是独立 Web App，不是浏览器扩展。与 Language Reactor 相比，约实现了 70-75% 的功能。

## ✨ 已实现功能

### 核心功能
- 🎥 **YouTube 视频播放器** - 嵌入式播放，字幕同步
- 📝 **双语字幕系统** - 原文+翻译并排显示，字幕面板，时间轴跳转
- 🔤 **单词翻译** - 点击单词即时翻译，保存到词汇表
- ⚡ **视频控制增强** - 自动暂停、循环播放、键盘快捷键、跳过已知句子
- 📚 **词汇管理系统** - 保存、管理、标记学习词汇
- 🎯 **SRS 间隔重复系统** - SM-2 算法，智能复习队列
- 🎧 **PhrasePump 听力练习** - 填空练习，音频播放，发音练习
- 📖 **智能文本阅读器** - 双语对照，点击单词即时翻译
- 🎬 **内容目录** - 精选 YouTube 学习资源
- 📺 **频道浏览** - 12 个精选学习频道，按语言和难度筛选
- ❤️ **视频收藏** - 管理喜欢的学习视频
- 🔍 **高级搜索** - 多维度筛选（分类、难度、时长、标签）+ 智能排序
- 🌐 **自动语言检测** - 识别视频语言并智能推荐语言对
- 💾 **Anki 导出** - 一键导出为 Anki 卡片格式
- 📊 **学习统计** - 追踪学习进度和成就系统
- 🏆 **成就徽章** - 13 个成就，7 个等级
- 🔥 **连续学习** - Streak 追踪，保持学习动力

### 技术特性
- 🌍 **多语言支持** - 中文、英文、日文界面（基于 next-intl）
- 🌐 **语言对选择** - 22 种语言对组合（英→中、日→英、韩→中等）
- 🤖 **智能语言检测** - 自动识别 8 种语言（英、中、日、韩、泰、西、法、德）
- 💡 **智能推荐** - 基于视频语言自动推荐最佳语言对
- 🔐 **本地认证系统** - 无需配置数据库即可使用（支持 Supabase 可选集成）
- 📱 **PWA 支持** - 可安装为桌面/移动应用
- 🎨 **现代化 UI** - 深色主题，响应式设计，WCAG AA 无障碍标准
- ✨ **流畅动画** - Framer Motion 驱动的过渡效果和交互动画
- ⚡ **性能优化** - 图片懒加载，虚拟滚动，字幕预加载

## ⚠️ 与 Language Reactor 的主要差距

### 架构差异
- ❌ **不是浏览器扩展** - 本项目是独立 Web App，无法注入到 YouTube/Netflix 页面
- ❌ **需要手动输入视频 ID** - 不能直接在 YouTube 页面上使用

### 缺失功能
- ❌ **AI 辅助功能** - AI 解释句子、Chatbot 对话练习（完全缺失）
- ❌ **多平台支持** - 仅支持 YouTube，不支持 Netflix/Vimeo
- ❌ **浏览器扩展** - 无法作为 Chrome/Firefox 扩展使用

详细功能对比请查看 [ROADMAP.md](./ROADMAP.md)

## 🚀 快速开始

### 基础安装

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 认证配置

项目支持两种认证模式：

#### 1. 本地模式（默认，无需配置）
- 用户数据存储在浏览器 localStorage
- 适合开发和个人使用
- 无需任何配置即可注册/登录

#### 2. Supabase 模式（可选）
如需云端数据同步，创建 `.env.local` 文件：

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

详细使用指南请查看 [USAGE.md](./USAGE.md)

## 🛠️ 技术栈

- **框架**: Next.js 15.0.7 (App Router) + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **国际化**: next-intl
- **状态管理**: Zustand
- **认证**: Supabase (可选) + 本地 fallback
- **翻译 API**: MyMemory Translation API
- **视频 API**: YouTube Data API v3 (可选)

## 📖 使用说明

1. **文本阅读**: 访问 `/app`，导入文本开始学习
2. **保存单词**: 点击任意单词查看翻译并保存
3. **听力练习**: 访问 `/phrasepump` 进行听力训练
4. **浏览内容**: 访问 `/catalog` 查看学习资源
5. **查看统计**: 访问 `/stats` 追踪学习进度和成就
