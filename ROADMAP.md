## 🚧 待完成功能

### 🔴 与 Language Reactor 的主要差距

#### 核心功能缺失
- [ ] **浏览器扩展架构** - 当前是独立 Web App，不是浏览器插件
  - 无法注入到 YouTube/Netflix 页面
  - 无法作为 Overlay 覆盖层使用
  - 需要用户手动输入视频 ID
  - **注意**: 实现浏览器扩展需要完全重构架构，工作量极大（数周）
  
- [ ] **Netflix 支持** - 受 DRM 限制
  - Netflix 使用 Widevine DRM 保护
  - 无法通过常规方式获取字幕和控制播放
  - 需要浏览器扩展 + 特殊权限
  - **注意**: 法律和技术限制使其难以实现

#### 已完成的功能
- [x] **AI 辅助功能** - 已实现基础版本
  - AI 句子解释（语法分析、词汇提取）
  - AI 对话练习（场景对话、难度分级）
  - 集成到字幕面板（Sparkles 按钮）
  - 支持配置真实 AI API（OpenAI/Anthropic）
  
- [x] **Vimeo 支持** - 已完整实现
  - Vimeo Player API 集成
  - 统一播放器接口
  - 自动平台检测
  - 与 YouTube 相同的控制功能

## 📝 开发原则

1. **用户至上**: 每个功能都要问"这对用户有什么价值？"
2. **快速迭代**: 先做 MVP,再优化
3. **数据驱动**: 用统计数据指导功能优先级
4. **代码质量**: 保持测试覆盖率,遵循最佳实践
