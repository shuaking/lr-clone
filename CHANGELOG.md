  - 搜索功能

## [2026-04-10] - 本地认证系统与质量优化

### 新增功能
- ✨ **本地认证系统**: 无需配置 Supabase 即可使用完整的注册/登录功能
  - 用户数据存储在浏览器 localStorage
  - 支持注册、登录、登出
  - 中间件路由保护
  - Cookie 认证状态同步
  - 自动 fallback 到 Supabase（如果已配置）

### 修复问题
- 🐛 **React 水合错误**: 修复主题切换导致的 hydration mismatch
  - 添加 `suppressHydrationWarning` 到 `<html>` 标签
  - 添加阻塞脚本在水合前应用主题
  - 重构布局结构，将 HTML 移至 `[locale]/layout.tsx`

- 🐛 **多语言切换失败**: 修复语言切换后内容不更新的问题
  - 将 `useTranslations` 改为 `getTranslations`
  - 在服务器组件中正确传递 locale 参数
  - 添加 `lang={locale}` 属性到 HTML 标签

- 🐛 **中间件认证检查**: 修复 cookie 解析逻辑
  - 正确解码 URL 编码的 cookie 值
  - 安全解析 JSON 数据
  - 准确检查用户对象是否存在

- 🐛 **Next.js 配置警告**: 修复 themeColor 配置位置
  - 将 `themeColor` 从 `metadata` 移至 `viewport` 导出

### 优化改进
- 🎨 **颜色对比度**: 优化品牌色和文本色以符合 WCAG AA 标准
  - 品牌色: `#7c9cff` → `#5b7dd4` (对比度 4.5:1)
  - 静音文本色: `#b4bcc9` → `#9ca6b8`
  - 所有颜色对比度测试通过

- 📝 **表单验证反馈**: 添加实时错误提示
  - 邮箱格式验证
  - 密码长度验证
  - 红色边框和错误消息
  - 输入时自动清除错误

- ♿ **无障碍改进**:
  - 添加 HTML `lang` 属性
  - 优化键盘导航
  - 改进屏幕阅读器支持

### 技术更新
- ⬆️ **Next.js**: 15.0.3 → 15.0.7
- ⬆️ **gstack**: 0.16.1.0 → 0.16.2.0
