# 翻译和词典功能测试计划

## 📋 功能概述

### 1. 翻译服务
**提供商**：
- DeepL (付费，需要 API key)
- Google Translate (付费，需要 API key)
- LibreTranslate (免费/自托管)
- MyMemory (免费)
- Mock (开发环境)

**特性**：
- 自动降级（主提供商失败时切换到备用）
- 缓存机制（1小时）
- 批量翻译支持

### 2. 词典服务
**提供商**：
- FreeDictionary API (免费)
- Tatoeba (例句数据库)
- Lemmatization (词形还原)

**特性**：
- 并行查询多个提供商
- 结果聚合
- 缓存机制（24小时）

### 3. UI 组件
- **ApiSettings** - API 配置界面
- **WordPopupEnhanced** - 增强的单词弹窗
- **SubtitleTranslationButton** - 字幕翻译按钮

## 🐛 已知问题

### 1. Mock Provider
**位置**: `lib/translation/providers/mock.ts`
**问题**: 开发环境默认使用 Mock，可能需要手动配置真实提供商
**解决**: 在 API 设置中配置首选提供商

### 2. API Key 存储
**位置**: `lib/stores/api-config-store.ts`
**问题**: API key 存储在 localStorage，不够安全
**建议**: 生产环境应该使用服务器端代理

### 3. 免费 API 限制
**问题**: 
- MyMemory: 每天 1000 次请求限制
- FreeDictionary: 无官方限制但可能被限流
- Tatoeba: 无官方 API，使用网页抓取
