# 快速开始指南

## 🚀 5分钟上手

### 1. 启动应用
```bash
# 应用已经在运行
访问 http://localhost:3000
```

### 2. 开始学习

#### 📖 使用文本阅读器
1. 点击导航栏的 "App" 或访问 http://localhost:3000/app
2. 确保 "Reader" 标签被选中
3. 粘贴任何英文文本到文本框
4. 点击 "开始阅读"
5. 点击任意单词查看翻译
6. 点击书签图标保存单词

#### 📚 管理词汇
1. 在应用页面点击 "Saved words" 标签
2. 查看所有保存的单词
3. 使用搜索框查找特定单词
4. 点击 "导出 Anki" 下载 CSV 文件
5. 导入到 Anki 进行复习

#### 🎧 听力练习
1. 访问 http://localhost:3000/phrasepump
2. 确保已保存一些单词
3. 点击播放按钮听音频
4. 在输入框填入听到的单词
5. 点击 "提交答案"
6. 查看结果并继续下一题

#### 🎬 浏览内容
1. 访问 http://localhost:3000/catalog
2. 使用分类按钮筛选内容
3. 选择难度级别
4. 使用搜索框查找感兴趣的视频

## 💡 使用技巧

### 文本阅读器
- **快速保存**: 点击单词后直接点击书签图标
- **重复查看**: 已保存的单词会在弹窗中显示绿色书签
- **语音朗读**: 点击喇叭图标听单词发音

### 词汇管理
- **批量导出**: 一次性导出所有单词到 Anki
- **搜索过滤**: 输入单词或翻译快速查找
- **查看语境**: 每个单词都保存了原始句子

### PhrasePump
- **多次播放**: 可以重复播放音频
- **即时反馈**: 提交后立即看到正确答案
- **进度追踪**: 顶部显示当前进度和得分

## 🎯 学习建议

1. **每天导入新文本**: 选择感兴趣的文章或书籍段落
2. **保存关键词汇**: 不要保存所有单词，只保存重要的
3. **定期练习**: 每天使用 PhrasePump 复习 10-15 分钟
4. **导出到 Anki**: 每周导出一次，使用间隔重复记忆

## 🔧 常见问题

**Q: 翻译不准确怎么办？**  
A: 当前使用免费 API，可能不够精确。未来可以切换到 DeepL 或 Google Translate。

**Q: 保存的单词丢失了？**  
A: 当前使用 localStorage，清除浏览器数据会丢失。建议定期导出到 Anki。

**Q: 语音播放没有声音？**  
A: 确保浏览器支持 Web Speech API，并且没有静音。

**Q: 如何添加更多内容？**  
A: 编辑 `lib/content-data.ts` 文件添加更多视频数据。
